Steps
=====

Set Up Monorepo
---------------

1. Create `atoll-mono` root folder.
2. `npm init`
3. `npm init -w packages/shared --scope atoll`
4. `npm init -w packages/web-app --scope atoll`
5. `npm init -w packages/desktop-app --scope atoll`
